Setting up your HTML document

In this lesson, we'll be teaching you the proper layout of an HTML page. You've probably already noticed in the previous lessons that every HTML page is set up in a similar fashion, with the <!DOCTYPE html> tag at the top of the page, followed by the opening HTML tag - <html> and the closing HTML tag - </html>, which enclose all of the other elements of the page. Check out Fig. 1 below:

Explanation of the HTML layout

As you can see in Fig. 1, the layout of an HTML page is divided into multiple sections. The opening <head> and closing </head> tags make up one section. This section contains the website's title, enclosed between the opening <title> and closing </title> tags. You can also add other info to this section too, such as styles, links, and scripts (will explain more about this stuff later). Below the <head> section is the <body section. This section encloses all of your website's visible content, such as headers and paragraphs. Both the <head> section and the <body> section are enclosed within the opening <html> and closing </html> tags.

And that, ladies and gentlemen, is the proper layout of an HTML page. Be sure to follow this layout when writing HTML - otherwise, your computer won't understand your code!
