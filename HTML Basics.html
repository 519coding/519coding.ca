An basic HTML document

Here is an example of a super-simple HTML document, just to give you an basic idea of what you'll be working with:

<!DOCTYPE html>
<html>
<head>
<title>This is our title!</title>
</head>
<body>
<h1>Here is a heading</h1>
<p>Here is a paragraph.</p>
</body>
</html>

HTML elements

All HTML documents are comprised of many different components, which serve as the building blocks of a website. These building blocks are referred to as elements. HTML elements can be used to create a variety of different things for your website (i.e. Headings, captions, blocks of text). The example document above contains many useful elements that are used in almost every website. We'll explain them below:

<!DOCTYPE html> - This is not techinically an element since it doesn't create any visible content on your website. We type this out at the top of our HTML document to let the computer know that we are using the HTML language.

<html> - This element is called the "root" element of an HTML page. All the other elements of an HTML document are sandiwched inside of this element. Just like the previous element we described, this element does not generate any visible content onto your webpage.

<head> - This element contains the title of your website, stylesheets, scripts (we'll get more into this later). Just like the two previous elements we described, this element does not generate anything onto your webpage.

<title> - This element is used to create a title for your website. The title of your website is displayed at the top of the browser tab, as shown below:

<body> - The <body> element contains the framwork for all of your website's visible content (i.e. Headings, paragraphs, sub-headings).

<h1> - This element is used to give your website headings. There are 6 heading sizes in html, numbered in descending order from <h1> all the way to <h2>, with <h1> being the largest heading and <h2>. Click the button below to see these headings in action:

<p> - The <p> is used to create paragraphs on your website. We'll talk more about this shortly.

HTML tags

All HTML elements are made up of tags. Tags usually come in pairs, like so <p> </p>. (Note that the second tag must have a forward slash → /. Kinda weird, but it's a rule. I guess it's so that you can distinguish between the two tags). The first tag is called the opening tag and the second tag is called the closing tag. Almost all HTML elements are represented by an opening tag followed by a closing tag, with the exception of special elements such as <!DOCTYPE html>. Most elements however, contain both an opening tag and a closing tag. Think of it like a sandwich. The two tags are like the bread, and the content between the bread is all the tasty stuff.

<p> → this is your sandwich's bread
This is the text that goes between your <p> and </p> tags. This text is actually visible on your website, the tags just tell the computer that you are making a pargagraph. This is like the ham and cheese of your sandwich - the tasty stuff!
