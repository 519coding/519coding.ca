Working with HTML - Tutorial for using Repl.it

In this lesson, we'll be teaching you how to use a website called Repl.it, a tool that allows you to create and edit your own HTML documents.

How to use Repl.it - Step by Step

➢ Step 1: Go to Repl.it and create an account.

➢ Step 2: Click on 'Start coding now'. From the list of languages, select 'HTML, CSS, JS', as shown below:

➢ Step 3: Write some HTML. The page has already been set up for you. See the sample document below for a quick refresher on how to incorporate headings, paragraphs, titles etc.

<!DOCTYPE html>
<html>
<head>
<title>This is our title!</title>
</head>
<body>
<h1>Here is a heading</h1>
<p>Here is a paragraph.</p>
</body>
</html>

➢ Step 4: Give your document a title by clicking the little ✎ icon at the top. Next, to see what you created with this html document, click the tiny icon, as indicated in the image below.

➢ Step 5: Celebrate the fact that you have successfully created a website with Repl.it!

Additional info about Repl.it & suggestions

➢ Repl.it automatically saves your work - you won't need to worry about losing your work.

➢ You can re-access and edit your work as long as you're logged into you repl.it account.

➢ You can access the main menu by clicking the repl.it logo in the top left corner. In the main menu, you can access your previous work as well as create new files.

We encourage you to practice the concepts you learn in each lesson on Repl.it
