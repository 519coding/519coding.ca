How to style your text

In the previous lesson, we did a brief overview about the different ways you can style your website's text. This lesson will serve as a comprehensive reference guide containing cool ways you can style your text and spice up your website.

Changing your text colour

We can change the colour of our website's text using the color property and assigning it to the name or rgb code of the color we want to change our text to: <p style="color:blue">Hello there</p> or <p style="color:rgb(0,0,255)">Hello there</p> (Note that HTML uses the American spelling of the word "colour", without the letter "u"). Changing your text size.

Changing your text font

You've probably noticed that HTML's default font looks pretty boring. Luckily, you can spice it up using the font-family property and assigning it to a font of your choice: <p style="font-family: Comic Sans MS">Hello there</p>. Click the button below to see how it's done:

Changing your text alignment

As you may have noticed, when we create a heading or paragraph in HTML, it is always aligned along the left sife of the page. But what if I want to keep my text on the center of the page or the right side of the page?. Well, to do that, you can use the text-align property and assign it to one of three values: left, right, or center. Click the button below to see how it's done:

Text formatting

All of the style properties from above are used together with the style attribute. However, that's not the only way we can style our text. Another way we can spice up our text is by using special elements called formatting elements, which can create special types of text. Here are some useful formatting elements:

➢ <b> - This element creates bold text. We sandwich our text between the opening <b> and closing </b> tags.

➢ <i> - This element creates italicized text. We sandwich our text between the opening <i> and closing </i> tags.

➢ <i> - This element creates italicized text. We sandwich our text between the opening <i> and closing </i> tags.

➢ <mark> - This element creates marked text. We sandwich our text between the opening <mark> and closing </mark> tags.

➢ <small> - This element creates small text. We sandwich our text between the opening <small> and closing </small> tags.

➢ <del> - This element creates deleted text. We sandwich our text between the opening <del> and closing </del> tags.

➢ <ins> - This element creates inserted text. We sandwich our text between the opening <ins> and closing </ins> tags.

➢ <sub> - This element creates subscript text. We sandwich our text between the opening <sub> and closing </sub> tags.

➢ <sup> - This element creates superscript text. We sandwich our text between the opening <sup> and closing </sup> tags.
