HTML elements and attributes

In this lesson, we'll be teaching you some cool attributes that you can use together with elements. As we mentioned in the previous two lessons, elements are made up of an opening tag and a closing tag → <head></head> with your content sandwiched in between these two tags. However, what you may not know is that we can still add additional info to these elements by assigning them attributes. You can assign attributes to certain elements by inlcuding them in the opening tag, like so: <p style="color:blue">Hello there</p> (the part highlighted in yellow represents the attribute and its assigned value). We'll explain this more thoroughly below.

The src attribute

Websites can look pretty boring without images. You can add images to your website by using the src attribute together with the <img> element. We can do this by setting the src attribute equal to the image's url or file name, like so:
<img src="https://c1.staticflickr.com/6/5140/5423307327_57cde2b5b7_b.jpg">.

The width and height attributes

Sometimes our image comes out too big, like this one. In that case, we have to resize it by adding the height and width attributes to our <img> element, like so: <img src="https://c1.staticflickr.com/6/5140/5423307327_57cde2b5b7_b.jpg" width="300" height="250">.

The alt attribute

Those who are visually impaired and have trouble seeing the website in front of them often use screen reader technology that reads the contents of the website aloud. However, when we have an image on our website, screen readers cannot express them out loud and thus, those who are visually impaired won't be able to understand what the image contains. To prevent this from happening, we add the alt attribute to our <img> element, like so:
<img src="https://c1.staticflickr.com/6/5140/5423307327_57cde2b5b7_b.jpg" alt="smiley face balloon"width="300" height="250">
An alt attribute, like the one above, should describe the contents of the image - If it's a picture of giraffe, don't describe it as a gorilla in your alt attribute ☺

The href attribute

The href attribute is used together with the <a></a> element to give your website a hyperlink. We can assign any link we want by setting it equal to the href attribute, like so: <a href="https://519coding.ca/">Click here<a>. Click the button below to get a better idea of what this looks like:

The target="_blank" attribute

The target="_blank" attribute is used together with the <a></a> element to create a hyperlink that opens in a new tab when clicked on. We add the target="_blank" attribute like this: <a href="https://519coding.ca/" target="_blank'>Click here<a>. Click the button below to see how this works:
