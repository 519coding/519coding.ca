String Literals

Strings are a unique data type in most programming languages. Integers and Floats are basic objects that aren't made up of anything else. However, Strings are a sequence of characters. For example, "Hello, World!" is made up of the characters "H", "e", "l", "l", "o", ",", " ", "W", "o", "r", "l", "d", "!" . This property of Strings allows us to do several interesting things with them.

String Concatenation

We can aslo combine different strings into one big string, using the + operator:
print("Hello" + ", " + "world!"). This is called string concatenation.

String Indices and Slices

In Python, you are able to retrieve specific characters from a string. All characters in a string are have an number index assigned to them. BUT! Instead of starting at 1, the index of the leftmost character is 0 and the last character is the length of the string minus 1. This is called a zero-based numbering system.

Suppose:

s = "Monty Python"

What if you wanted to use only the first character of that string, "M"?

Doing this is actually quite simple. To get a character from a string, type the variable name with set of square brackets after the variable name. The square brackets should contain the index of the character you want to retrieve.

You can also retrieve a Slice of a string. A slice is just a string contained within a string.

The square bracket now takes two arguments that are separated by a colon. The first value is the index of the starting character, the second value is the index of the ending character PLUS ONE!!!!! NOTE! The first argument is INCLUSIVE the second argument is EXCLUSIVE.

If the value before or after the colon is left empty, then the index will be set to either the start or the end of the string.

What if you wanted to have every second character in a string? Well, the square brackets can take a third value, also separated by a colon, called the step.

In the last two examples you can see that when the step is negative, you get a string that is reversed. NOTE! When the step is negative, then the starting index must be the higher index, and the ending index must be the lower index.
