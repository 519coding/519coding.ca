What in the world is a string?

No, we are not referring to a piece of yarn that your cat plays with. Strings are a unique and useful data type in Python. In our previous lessons, you have already seen strings being used in our print() functions: print("Hello World").

Strings are used to produce text. We tell the computer that we are working with a string by enclosing our text between either a pair of double quotation marks → " " or single quotation marks → ' '. The text we enclose between these quotation marks is output to the compiler using the print() function.

String Literals

Strings are a unique data type in most programming languages. Integers and Floats are basic objects that aren't made up of anything else. However, Strings are a sequence of characters, which may include letters, spaces, numbers, or punctuation marks/symbols. For example, "Hello World!" is made up of 12 characters (10 letters, 1 space, 1 punctuation mark). This property of Strings allows us to do several interesting things with them. We'll talk about them below:

String Concatenation

One of the cool things we can do with strings is concatenate them. Concatenate is just a fancy word for combining things together. We combine/concatenate different strings into one big string using the + operator: print("Hello" + ", " + "world!"). This is called string concatenation. Check out the code below to see this in action:

String Indices

Another cool thing you can do with strings is retrieve specific characters from them. All characters in a string have an number index assigned to them. BUT! Instead of starting at 1, the index of the leftmost character is 0 and the last character is the length of the string minus 1. This is called a zero-based numbering system.

Let's say you only want to print the first letter of a string:

Suppose:

Doing this is actually quite simple. To get a specific character from a string, type the variable name with set of square brackets after the variable name. The square brackets should contain the index of the character you want to retrieve.

Try it yourself in the editor below:

Write your print() function in the line below: a = "Python" print(a[4]) test_function("print") success_msg("You did it!")
The third character of the string is represented be index [4].

Trickier slices

What if you wanted to have every second character in a string? Or maybe you want to print letters of your string in reverse order? Well, the square brackets can take a third value, also separated by a colon, called the step.

NOTE! When the step is negative, then the starting index must be the higher index, and the ending index must be the lower index. For the example above, the starting index is 9 and the ending index is 1.

Try slicing strings yourself in the editor below:

# Use a print() function to print out every second character of the word "Python" a = "Python" # The letters 'Pto' should be output to the compiler. Write your print() function in the line below: a = "Python" print(a[::2]) test_function("print") success_msg("You did it!")
The square bracket should look like this [::2] with the first two values left blank and the "step" value equal to 2.
